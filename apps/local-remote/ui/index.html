local-remote

Connect to the platform Redis, listen for SYSTEM:* events, and send OSC over UDP when SEGMENT_ACTIVATED matches your room and segment map (e.g. trigger a Farrago tile).

Connection
OSC (SEGMENT_ACTIVATED)

Use 127.0.0.1 when Farrago runs on this Mac.

Must match OSC Input port in Farrago Settings → Controllers.

Sends /ping to the host and port above (see Farrago OSC). Success = UDP send OK (see bytesSent). Farrago often does not reply back to this app—it sends responses to OSC Output, so replyReceived: false is common and not proof of failure. Same Mac as Farrago: try 127.0.0.1. For tiles, use default args (e.g. 1) with /set/…/play or /select. No save required for this button.

Sent as float32 after the address on every trigger (many Farrago endpoints expect a value).

Full path starting with /, e.g. /set/selected/tile/0/0/play
